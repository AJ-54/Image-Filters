## Introduction

This project aims towards making Image filters using the python OpenCV library that can be used as filters for an photo editing application.
Some basic filters include : 
- black and white
- sketch 
- cartoonizer

## To install
The language used is Python3
  
  ### Required libraries
  ```sh
      OpenCV
      
      pip install opencv
```
